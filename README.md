# School Stock Management System Frontend

A production-style ReactJS frontend for a school stock management dashboard.

## Tech
- ReactJS (Create React App)
- React Router
- Tailwind CSS CDN
- Context API
- localStorage
- Recharts
- Lucide React

## Run
```bash
npm install
npm start
```

## Notes
- This project is frontend-only.
- All data is stored in localStorage.
- The app uses school and inventory terms only.
- Tailwind is loaded from the CDN in `public/index.html`.
